Slide header

This slide is in a custom `slide` element. Most browsers in use will allow new elements to be used in html documents. The developer is just responsible for defined all of the styling for it. There are no defaults to fall back on. Which can be use in some cases, because it is less likely that some other browser will handle it differently

  • Not generally recommended, but the slide show supports it
  • Does not even need any special setup
  • Leave the `data-selector` attribute off of the slide show element, and the defaults will find it
